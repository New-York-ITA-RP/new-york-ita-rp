# New York ITA RP - Sito ERLC

File separati:
- `index.html` = struttura del sito
- `css/style.css` = grafica blu, responsive e animazioni
- `js/script.js` = menu mobile, membri totali e versione
- `assets/backgrounds/` = metti qui gli sfondi
- `assets/fazioni/` = metti qui le immagini delle fazioni

## Come cambiare immagini
Inserisci immagini con questi nomi:

### Sfondi
- `assets/backgrounds/hero.jpg`
- `assets/backgrounds/story.jpg`

### Fazioni
- `assets/fazioni/procura.jpg`
- `assets/fazioni/ice.jpg`
- `assets/fazioni/nypd.jpg`
- `assets/fazioni/nyso.jpg`
- `assets/fazioni/border-patrol.jpg`
- `assets/fazioni/nyfd.jpg`
- `assets/fazioni/ems.jpg`
- `assets/fazioni/dot.jpg`
- `assets/fazioni/aziende.jpg`
- `assets/fazioni/gang.jpg`
- `assets/fazioni/mafie.jpg`

## Come cambiare membri e versione
Apri `js/script.js` e modifica:

```js
members: 250,
version: "v1.0.0"
```
